1. El Dataset

Utilizamos el dataset Auto MPG del repositorio UCI Machine Learning. Recopila datos de vehículos fabricados en EE.UU., Europa y Japón (1970-1982), originalmente recolectados para estudiar el consumo de combustible.

2. Selección de Variables

Variable Dependiente (Y)

mpg (Millas por galón)

Es lo que queremos predecir: cuánta gasolina consume el carro.

Independientes (X)

horsepower weight displacement

Se eligieron porque tienen una relación física directa con el consumo y son continuas, ideales para regresión lineal.

3. Preprocesamiento de Datos

Datos Nulos

Se eliminaron las filas sin datos completos en horsepower.

Duplicados

No se encontraron datos duplicados en el set de datos.

Categóricas

Todas las variables elegidas son numéricas, sin necesidad de encoding.

Outliers

Se detectaron y analizaron datos anómalos usando el método de Rango Intercuartílico (IQR).

Posteriormente, los datos numéricos fueron estandarizados mediante StandardScaler de scikit‑learn, garantizando media 0 y varianza 1, lo que mejora la estabilidad del algoritmo de regresión lineal.

División del Dataset

Para entrenar y evaluar el modelo se utilizó train_test_split con una proporción 80 % entrenamiento / 20 % prueba y random_state=42 para garantizar reproducibilidad. Esta separación permite estimar el desempeño del modelo en datos no vistos y evitar sobre‑ajuste.
